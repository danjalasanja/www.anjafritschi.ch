Concept

Chess... a very complex strategy game. With the help of hand tracking, you can learn this game easily.
Various tools and analyses improve your chess skills. With a beamer, the learning graphic is projected onto the chess board and can react directly to moves and give hints. The game provides statistics in which the learning curve can be checked and compared with other players.
