Density Destiny

AUTUMN 2022
interactive poster design
‍
Groupwork By
Anja, andy

Concept

This interactive poster design is based on the anagram "density destiny". With a camera that tracks you as a blob, it is possible to exert influence. The poster was exhibited in a store in Zurich. Whenever someone walks by, it moves.

Process

187 drafts before...
The first designs were very different if you compare them. we were inspired by geometric but also organic shapes. In the second design phase, we were quite quickly drawn in the direction of metaballs-inspired shapes.

With the help of some support in code and debugging, we created our own code for the interactive poster. With the p5.js library, we programmed the movements in the visual studio code. It was important to us that the design also moves when no one is standing in front of it. So we had to program different states.
